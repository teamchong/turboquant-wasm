turboquant-wasm

Shrink embedding vectors 6x. Search them without decompressing.

Vector compression has existed for years (FAISS, ScaNN, PQ), but they require a training step on your dataset first. TurboQuant (Google Research, 2024) skips that — just encode any vector immediately, no codebook, no training. Three lines of code: encode(), decode(), dot(). This npm package runs it in the browser via WASM with relaxed SIMD.

🔍
Vector Search
Type a query, search 5K Wikipedia passages instantly — all in your browser. Left column uses compressed vectors (1.2 MB), right uses uncompressed (7.7 MB). Same results, 6x less data to download.
